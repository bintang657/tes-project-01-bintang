# Belajar Kelas SMA

Belajar Kelas SMA adalah platform pembelajaran mandiri berbasis web
yang ditujukan untuk siswa SMA kelas 10â€“12 (IPA dan IPS).

## Fitur
- Materi terstruktur kelas 10, 11, dan 12
- Materi IPS (Sejarah, Ekonomi, Geografi)
- Contoh soal dan pembahasan
- Kuis interaktif
- Checklist belajar tersimpan
- Dark dan light mode
- Siap digunakan di GitHub Pages

## Tujuan
Membantu siswa SMA memahami materi pelajaran
sekaligus mengenal teknologi web dan GitHub sejak dini.
